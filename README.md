# DESCRIPCIÓN DEL PROYECTO JUEGO DE LA OCA

El Software que crearemos estará basado en el Juego de la Oca,
un juego ideado para que, dentro de un margen de 64 casillas,
el profesor pueda ingresar preguntas para éstas 64 casillas,
donde los jugadores (estudiantes) tendrán que realizar equipos
y lanzar dados para intentar llegar a la casilla 64,
donde responderán la última pregunta y si la responden
correctamente acabará el juego. Además, según lo hemos previsto,
la forma del escenario se podrá cambiar, también se podrá cambiar
la cantidad de casillas y por ende la cantidad de preguntas a realizar
(ya que es una por casilla) también cambiaría, aparte de que cada pregunta
tendrá su respectivo castigo como penalización por responder incorrectamente,
se avanzará por el escenario de juego las casillas que indique el dado lanzado
al inicio del turno de cada equipo, y si el equipo está próximo a ganar el juego,
tendrá que sacar el número exacto de casillas para llegar a la 64
con el fin de responder la última pregunta y ganar la partida.
